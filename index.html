Tjänster vi erbjuder

Vi är ett enmansföretag som siktar på att skapa webbsidor och program som tillfredställer. På grund av vår produktivitet erbjuder vi webbsidor som imponerar och program som aldrig besviker. Vi gillar att säga att vi tar oss ann vilka utmaningar som än kommer, därför begränsar vi inte vår målgrupp särkilt mycket. De primära personerna vi vänder oss till är företag samt andra enmansföretag men vi erbjuder även våra tjänster till privatpersoner som önskar att marknadsföra sitt arbete. Eftersom vi för tillfället har mest kunskaper inom HTML och CSS så är vår specialiteter webbdesign, dock jobbar vi hårt för att kunna erbjuda komplicerade program byggda med till exempel C#, Java och eventuellt JavaScript.
